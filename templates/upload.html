Upload CSV File
The CSV file must contain player_nickname and net columns.
CSV Format Requirements:
  • player_nickname: Player's name (required)
  • net: Net profit/loss for the session in cents (required) - will be automatically converted to dollars
  • Other columns like player_id, session_start_at, session_end_at, buy_in, buy_out, stack are optional
Important Note:

CSV values are expected to be in cents (e.g., 5000 = $50.00). The system will automatically convert them to dollars.
